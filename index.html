SDND - Simple DragNDrop

SDND is a very lightweight JavaScript & jQuery based plugin that allows you to make any HTML element draggable (and droppable functionality will be added soon) with a simple line of code.

You can add callback functions and lot more. Head over to the project page on GitHub for more details or download and try it out yourself. It is free to use and released under MIT license. Enjoy :)

Draggable sample 1

Desctiption:

Drag/move element around the screen with a simple single of line code. Supports standard jQuery selector.

Code:

new SDND('.draggable.a').init();

Draggable sample 2

Desctiption:

Drag by a child element (handler) and reverts to original position.

Code:

var drag2 = new SDND('.draggable.b');
drag2.options.elHandler = '.handler';
drag2.options.revert = true;
drag2.init();

Draggable sample 3

Desctiption:

Calls user defined functions during dragStart(), dragging() and dragEnd()

Code:

var drag3 = new SDND('.draggable.c');
drag3.options.revert = true;
drag3.callbacks = {
   dragStart : function() { drag3.currentEl.text('dragStart event called'); },
   dragging : function() { rag3.currentEl.text('dragging event called'); },
   dragEnd : function() { drag3.currentEl.text('dragEnd event called'); }
};
drag3.init();

Draggable sample 4

Desctiption:

Preserves original element and drags a clone. Keeps visible on screen.

Code:

var drag4 = new SDND('.draggable.d');
drag4.options.moveOriginal = false;
drag4.options.revert = false;
drag4.options.fixedPosition = true;
drag4.init();
